# Ditar-me-z-
Ditar me zë është një mënyrë personale për të regjistruar mendimet, ndjenjat dhe përvojat pa shkruar. Thjesht flet me zërin tënd për ditën, emocionet ose kujtimet. Të ndihmon të kuptosh veten, të reflektosh mbi ndjenjat dhe ta dëgjosh më vonë për të parë ndryshimet me kalimin e kohës.
